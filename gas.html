Pump setup

  1. Open reader at pump (key required)
  2. Turn off power
  3. On "Switch Two", press toggle "No6" down
  4. Turn power on
  5. Select "No" on options screen until you reach "program chip key"
  6. Select "Yes" when you read "program chip key"
  7. Select "Yes" for "System Format 2"
  8. Type in "0700"- Click "Enter"
  9. Type in thirteen 0's then enter key number (0000000000000xx) - There should be a total of 15 digits with the 0's and the two-digit key number
  10. Select "No" for "Odometer" reading
  11. Set the limit to 0
  12. Insert key
  13. Remove key when prompted
  14. Repeat steps 1 and 2
  15. On "Switch Two", press toggle "No6" up
  16. Turn power on
  17. Close and lock reader

Computer setup

  1. Open "Phoenix"
  2. Select "Site Information"
  3. Click on "Cards"
  4. Select Account
  5. Set card type to "Single"
  6. Click "New"
  7. Enter account number (0700) followed by thirteen 0's and the key number (07000000000000000xx)
  8. Name the account
  9. Setup PIN
  10. Update card
  11. Poll site
  12. Make sure the final validation screen appears with card number and PIN
